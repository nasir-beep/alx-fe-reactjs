# Form Handling in React (Controlled Components & Formik)

## Project Overview

This project demonstrates two approaches to handling forms in React:

1. Controlled Components using `useState`
2. Formik with Yup validation

The app includes a simple user registration form with fields for:
- Username
- Email
- Password

---

## Technologies Used

- React (Vite)
- Formik
- Yup

---

## Learning Objectives

- Manage form state manually using controlled components
- Implement basic validation logic
- Use Formik for advanced form handling
- Apply Yup schema validation

# Advanced Data Handling with React Query

## Project Overview

This project demonstrates advanced data fetching and caching using React Query.

Data is fetched from:
https://jsonplaceholder.typicode.com/posts

---

## Technologies Used

- React (Vite)
- @tanstack/react-query
- JSONPlaceholder API

---

## Learning Objectives

- Fetch data using `useQuery`
- Handle loading and error states
- Implement caching with `staleTime`
- Trigger manual refetching
- Reduce unnecessary API calls

---

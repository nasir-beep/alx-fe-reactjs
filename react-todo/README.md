# React Todo List App

## Project Overview

This project demonstrates a fully functional **Todo List** component in React.  
Users can:

- View initial todos
- Add new todos
- Toggle completion status
- Delete todos individually

Additionally, the project includes **unit tests** to verify all functionalities using Jest and React Testing Library.

---

## Technologies Used

- React (Vite)
- Jest
- @testing-library/react
- @testing-library/jest-dom
- @testing-library/user-event

---

## Learning Objectives

- Manage dynamic state with `useState`
- Implement component communication (`AddTodoForm` → `TodoList`)
- Handle user interactions (add, toggle, delete)
- Write comprehensive tests for React components
- Simulate user actions with `fireEvent` and `userEvent`

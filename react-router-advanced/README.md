# Advanced Routing in React with React Router

## Project Overview

This project demonstrates advanced routing techniques using React Router:

- Nested Routes
- Dynamic Routes
- Protected Routes
- Authentication simulation

---

## Technologies Used

- React (Vite)
- react-router-dom

---

## Learning Objectives

- Implement nested routes using `<Outlet />`
- Create dynamic routes using URL parameters
- Build protected routes with authentication checks
- Handle redirects using `<Navigate />`
